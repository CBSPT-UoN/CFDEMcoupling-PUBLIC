forceModel_particleVolume command :h3

[Syntax:]

Defined in couplingProperties dictionary.

forceModels
(
    particleVolume
);
particleVolumeProps
\{
    verbose switch1;
    writeToFile switch2;
    scale scalar1;
    startTime scalar2;
\}; :pre

{switch1} = (optional, default false) switch for output to screen :ulb,l
{switch2} = (optional, default true) switch for output to file :l
{scalar1} = (optional, default 1) scaling of the particle volume d=dSphere/scale :l
{scalar2} = (optional, default 0) start time of volume calculation and output :l
:ule

[Examples:]

forceModels
(
    particleVolume
);
particleVolumeProps
\{
    writeToFile false;
\} :pre

[Description:]

This "forceModel" does not influence the particles or the simulation - it is a postprocessing tool! The total volume of the particles located in the CFD domain is calculated.

[Restrictions:]

None.
